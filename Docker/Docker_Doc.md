# Official Docker References

For learning Docker, official documentation, and best practices, these are the trusted sources:

- **Docker Official Site** – [https://www.docker.com/](https://www.docker.com/)  
  *The official hub for Docker products, updates, and resources.*

- **Docker Documentation** – [https://docs.docker.com/](https://docs.docker.com/)  
  *Comprehensive guides and references for Docker commands, images, containers, and best practices.*

- **Docker Hub** – [https://hub.docker.com/](https://hub.docker.com/)  
  *Official registry for Docker images. Find verified images for Python, Node.js, Redis, and more.*

- **Docker Compose Documentation** – [https://docs.docker.com/compose/](https://docs.docker.com/compose/)  
  *Official guide for multi-container orchestration using Docker Compose.*

- **Docker GitHub Repository** – [https://github.com/docker](https://github.com/docker)  
  *Source code, releases, and community resources for Docker.*

## Want to Learn More?  

If you want to dive deeper into Linux and become more confident with commands and scripting, these are my go-to official resources:  

- **Linux Kernel Official Site** – [https://www.kernel.org/](https://www.kernel.org/)  
  *The official hub for Linux updates, releases, and kernel documentation.*  

- **Linux Documentation Project (TLDP)** – [https://www.tldp.org/](https://www.tldp.org/)  
  *Packed with guides, HOWTOs, and tutorials for every Linux topic you can imagine.*  

- **Linux Command Reference** – [https://linuxcommand.org/](https://linuxcommand.org/)  
  *A simple, practical reference to help you master Linux commands and daily tasks.*  

- **Bash Scripting Guide** – [https://www.gnu.org/software/bash/manual/](https://www.gnu.org/software/bash/manual/)  
  *The official GNU manual to level up your Bash scripting skills and automation.*
